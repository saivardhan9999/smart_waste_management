## Slide 3: Solution Overview
**AI-Powered Smart Waste Management System**
- Real-time waste classification using deep learning
- Interactive campus mapping with cleanliness levels
- Severity assessment for targeted cleanup
- Modern web interface with explainable AI

---

## Slide 4: Tech Stack
**Backend**: Python, Flask, SQLAlchemy, TensorFlow, Keras
**Frontend**: HTML5, CSS3, JavaScript, Bootstrap, Leaflet.js
**ML**: MobileNetV2, Transfer Learning, Grad-CAM
**Database**: SQLite with location intelligence

---

## Slide 5: Machine Learning Algorithm
**Architecture**: MobileNetV2 Transfer Learning
- **Input**: 224x224x3 RGB images
- **Classes**: 6 waste types (Glass, Paper, Cardboard, Plastic, Metal, Trash)
- **Dataset**: 2,527 training images
- **Accuracy**: 89%+ on validation set
- **Innovation**: Grad-CAM explainable AI
